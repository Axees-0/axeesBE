POST /api/v1/analyze/investment

Core investment analysis endpoint that replicates your Streamlit prototype's main functionality with enhanced AI capabilities.

Request Body
{
  "company_symbol": "NVDA",
  "investment_amount": 1000000,
  "time_horizon": "12_months",
  "risk_tolerance": "moderate",
  "analysis_depth": "comprehensive",
  "include_sentiment": true,
  "market_conditions": {
    "sector": "technology",
    "current_volatility": 0.25
  }
}
Response Body
{
  "analysis_id": "inv_analysis_12345",
  "risk_score": 6.2,
  "confidence_level": 0.94,
  "recommendation": "BUY",
  "target_price": 875.00,
  "key_insights": [
    "Strong Q4 earnings momentum",
    "AI market expansion driving growth",
    "Manageable debt-to-equity ratio"
  ],
  "ai_agent_analysis": {
    "sentiment_score": 0.76,
    "technical_indicators": "bullish",
    "fundamental_strength": 8.1
  }
}
Python Integration
import requests
import json

# API endpoint
url = "https://api.rominaday.ai/v1/analyze/investment"

# Request payload
payload = {
    "company_symbol": "NVDA",
    "investment_amount": 1000000,
    "time_horizon": "12_months"
}

# Make request
response = requests.post(url, json=payload)
analysis = response.json()

print(f"Risk Score: {analysis['risk_score']}")
print(f"Recommendation: {analysis['recommendation']}")
